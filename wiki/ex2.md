# try
```dart
try {
    do something...
} on 特定的Exceptions/Errors{
    //遇到特定例外時執行的部分
}catch(e){ 
    // 當你不確定可能引發的錯誤種類時可以使用catch
    // 它會接收所有的錯誤訊息
}finally{
    //　無論是否有遇到錯誤都會執行到的部分
}
```
